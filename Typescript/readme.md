# Typescript

## Compilar archivos de ts a js

```bash
  # Generar un archivo app.js
  tsc app.ts
```

## Creando archivo de configuración typescript

```bash
  # Genera un archivo tsconfig.js
  tsc --init
```

### Modo Observador o Watch
```bash
  # Compila automáticamente
  tsc -w
  # o
  tsc --watch
```
